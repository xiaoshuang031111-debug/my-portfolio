# Lorraine Portfolio

Welcome to Lorraine's channel.

This is a personal portfolio website for AI brand and growth operation work, built with React and Vite.

## Local Preview

```bash
pnpm install
pnpm dev
```

Then open:

```text
http://127.0.0.1:5173/
```

## Build

```bash
pnpm build
```

The production files will be generated in `dist/`.
